# Deployment Instructions

## The Problem
Your frontend calls `localhost:3001` which doesn't exist in production.

## The Solution
Set environment variable `VITE_API_URL` to your backend URL.

## Quick Fix for Vercel/Netlify

### Step 1: Deploy Backend
Deploy your backend separately (Vercel, Render, Railway) with these env vars:
- `LAMATIC_API_KEY`
- `LAMATIC_API_URL`
- `LAMATIC_WORKFLOW_ID`
- `LAMATIC_PROJECT_ID`

### Step 2: Configure Frontend
In your frontend deployment (Vercel/Netlify), add:
```
VITE_API_URL=https://your-backend-url.com
```

### Step 3: Redeploy
Redeploy your frontend after adding the environment variable.

## Example for Vercel

1. Deploy backend: `vercel --prod` (in backend folder)
2. Get URL: `https://traq-backend.vercel.app`
3. Add to frontend env: `VITE_API_URL=https://traq-backend.vercel.app`
4. Redeploy frontend

Your app will now work in production! 🚀
