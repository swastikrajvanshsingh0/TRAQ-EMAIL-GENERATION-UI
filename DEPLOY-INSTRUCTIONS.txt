
## Deploy to Netlify

### Step 1: Deploy
```bash
netlify deploy --prod
```

### Step 2: Add Environment Variables

In Netlify dashboard → Site Settings → Environment Variables, add the same 4 variables above.

### Step 3: Redeploy

Trigger a new deployment in Netlify dashboard.

## That's It! 🎉

Your app will work perfectly now. The frontend calls Lamatic API directly - no backend server needed!

## Troubleshooting

**Build fails?**
- Make sure you're deploying from the latest commit
- Check that all 4 environment variables are set
- Redeploy after adding variables

**API not working?**
- Verify environment variables are correct
- Check they all start with `VITE_` prefix
- Redeploy after changes
